# 🚀 What is Sliding Window?

Sliding Window is an optimization technique used mainly on:
- Arrays
- Strings
- Subarrays
- Substrings

Instead of recalculating values repeatedly for every subarray/substring, we:
- reuse previous computations
- add new elements
- remove old elements

This reduces time complexity from:

O(n²) → O(n)
